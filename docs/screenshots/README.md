# Screenshots

This folder contains screenshots used by the main `README.md`.

Recommended filenames:
- `control.png`
- `control-ams.png`
- `jobs.png`
- `overview.png` (optional)
- `mobile-control.png` (optional)

Safety:
- Do not include sensitive data in screenshots (printer IP, serial, access code).
- If you capture real jobs/printers, blur identifiers before committing.
